###Требования к проекту:

- Весь код в asmdef
- Использовать TMPro вместо обычного Text
- Использовать полученные знания с предыдущих занятий 
- Вести полноценный гит с коммитами
- Доп плагины не использовать, кроме тех что есть в проекте (tmpro разрешен через UPM)
- Не фокусироваться на красоте или эффектности арта, достаточно обычного текста, но поддержка анимаций интерфейса должна быть

####Цель: сделать достаточно гибкую игру, которую можно развивать и расширять с применением паттернов в тех местах где это действительно нужно и в том виде в котором это оправдано.
Вопросы вне митинга - в общий тред в Teams


Доп условие: имеющийся код в проекте можно менять в любом обьеме
